= Development =

The source code for the CRM shell is kept in a Mercurial source
repository hosted at +nongnu.org+. To obtain a copy of the source
code, you will need the Mercurial command line tool, +hg+:

    hg clone http://hg.savannah.nongnu.org/hgweb/crmsh/

For more information about how to use Mercurial and distributed
version control software in general, see the Mercurial website at
http://mercurial.selenic.com[mercurial.selenic.com].

`crmsh` is a free software project, and is open to
contributors. Patches, comments, documentation, testing and so on are
all very much welcome!
