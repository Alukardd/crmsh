= crm shell

== Introduction

The `crm` shell is a command-line interface for High-Availability
cluster management on GNU/Linux systems. It simplifies the
configuration, management and troubleshooting of
http://clusterlabs.org[Pacemaker]-based clusters, by providing a
powerful and intuitive set of features.

For more information and installation instructions, see the
link:documentation.html[Documentation] page.
